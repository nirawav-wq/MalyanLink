# MalyanLink 
**Developed by:** PURPLNOVA  
**Hardware:** ESP8266EX + ILI9488 (Recycled Malyan M300/M320 UI Board)

MalyanLink is a high-performance system monitor interface that revives old 3D printer displays.

##  Hardware Pinout
| Signal | ESP8266 GPIO | Malyan PCB Label |
| :--- | :--- | :--- |
| **CS** | 15 | LCD_CS |
| **RST** | 2 | LCD_RST |
| **MOSI** | 13 | LCD_SDI |
| **CLK** | 14 | LCD_SCK |
